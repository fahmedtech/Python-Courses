4.    [25 pts] Write function search() that takes a string url, a dictionary d and a number n and then crawls through all the web pages that can be reached, up to depth n,  from the web page named url. At the end of the crawl, the dictionary d should contain (key: value) pairs where the keys are the different links found through the crawl and the value for a link is the number of occurences of the link in all the web pages visited.

Usage:
>>> d = {}
>>> search('http://reed.cs.depaul.edu/lperkovic/csc242/test1.html', d, 1)
>>> for key in d:
    print(key, d[key])
