4.   Recall that that decimal (base-10) representation of non-negative integers is just one possible representation. In fact, integers are represented in binary (base-2) inside the computer. So, for example, decimal 9 is binary 1001, i.e. (9)10 = (1001)2. Here are the first few non-negative integers: (0)10 = (0)2,   (1)10 = (1)2,   (2)10 = (10)2,  (3)10 = (11)2,  (4)10 = (100)2, etc.Your task is to write recursive function numOnes() that returns the number of 1's in the binary representation of an integer N. Use the fact that this is equal to the number of 1's in the representation of N/2(integer division), plus 1 if N is odd.

Usage:
>>> numOnes(0)
0
>>> numOnes(1)
1
>>> numOnes(3)
2
>>> numOnes(15)
4
